def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

Useful Tips

  • Set draft: true in frontmatter to hide posts
  • Use #| echo: false to hide code but show output
  • Use #| eval: false to show code but not run it
  • Add line numbers with #| code-line-numbers: true
